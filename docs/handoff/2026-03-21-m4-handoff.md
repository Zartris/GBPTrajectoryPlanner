### This session — Critical bug fixes
- **3D Jacobian fix** — replaced broken arc-length Jacobian with proper 3D geometry:
  `jacobian_a = (pos_a - pos_b) · tangent_a / dist_3d`, `jacobian_b = -(pos_a - pos_b) · tangent_b / dist_3d`
  Both robots' tangents computed from map. Asymmetric dampening: `FRONT_DAMPING=0.3` (front robot gets 30% push, behind robot gets full push).
- **Stale anchor mean fix** — `variables[0].eta/lambda` now set directly at step 0, not just `prior_eta/prior_lambda`. Ensures `variables[0].mean()` is correct before `update_dynamics_v_nom()` reads it.
- **Static mut data race fix** — `TICK_COUNT` moved from `static mut [u32; 2]` (UB) to per-runner `tick_count: u32` field.
- **Silent IR insert fix** — `ir_set.insert()` now returns `bool`; if collection is full, orphaned factor is removed from graph.

### This session — New factor: VelocityBoundFactor
- **BIPM-inspired velocity bounding** (`crates/gbp-core/src/velocity_bound_factor.rs`)
  Pairwise factor on `(s_k, s_{k+1})` that enforces `v_min ≤ velocity ≤ v_max` using adaptive precision. Based on Barrier Interior Point Method (arxiv.org/abs/2506.14341) — simplified to fixed-kappa without outer loop. See module doc for full background.
  - Upper bound: `v_max` from edge speed profile (prevents 10+ m/s GBP overshoot)
  - Lower bound: `v_min = -0.3 m/s` (prevents belief chain backwards-flip while allowing slow creep-back for merge overshoot recovery)
  - Adaptive precision: 0 when well within bounds, grows as `1/(kappa * g²)` near boundary, caps at 100
  - K-1 permanent factors spawned alongside dynamics factors

### This session — Internal/external iteration split
- **`iterate_split(n_internal, n_external)`** in `FactorGraph` — separates message passing:
  - Internal factors (Dynamics, VelocityBound) get `n_internal` iterations per external round
  - External factors (InterRobot) get 1 iteration per external round
  - `is_internal()` method on `FactorKind` classifies factors
  - Matches MAGICS architecture (AU-Master-Thesis/magics) of TI/TE separation
  - Current tuning: `GBP_INTERNAL_ITERS=3`, `GBP_EXTERNAL_ITERS=5`

### This session — IR factor lifecycle fix
- **Per-variable IR factor despawn** — factors now removed when 3D distance at that timestep exceeds `IR_ACTIVATION_RANGE`. New `remove_single(robot_id, k)` on `InterRobotFactorSet`. Previously factors persisted forever once spawned, causing stale long-range links.

### This session — Visualiser improvements
- HUD now shows `cmd_v`, `gbp_v`, `dist3d` per robot
- `raw_gbp_velocity` and `min_neighbour_dist_3d` fields added to `RobotStateMsg`/`RobotState`
- Belief spread diagnostic (`max - min` of variable means) in `StepOutput`
- OSCILLATION warning when belief spread > 5m (throttled to 2Hz)

### This session — Simulator improvements
- **Collision detector** in distance monitor (10Hz):
  - Detects chassis overlap (dist < 1.15m center-to-center)
  - Throttled logging (1Hz during ongoing collision)
  - Spawn exclusion (ignores overlap when either robot near s=0)
  - Collision count in DIST log line
- **R1 trajectory shortening** — when both robots share goal, R1's trajectory last edge is shortened by d_safe so v_nom taper decelerates correctly (previously physics cap was misaligned with GBP trajectory)
- Removed unused `IR_RAMP_START` constant

## What's Fixed (previously broken)

### Jacobian — FIXED
3D Jacobian with tangent vectors replaces broken arc-length subtraction. Merge scenario no longer has pass-through. Both robots' tangents used for Schur complement coupling.

### GBP velocity overshoot — FIXED
VelocityBoundFactor caps raw GBP velocity. Before: `gbp_v=11+ m/s`. After: `gbp_v=3-4 m/s`. `DynamicConstraints` clips to `cmd_v=2.5 m/s`.

### Belief chain backwards-flip — FIXED
Lower velocity bound (`v_min=-0.3`) prevents IR factor from pushing all variables backwards. Per-variable IR despawn prevents stale factors from linking to distant positions.

### IR factor amplification — FIXED
Internal/external iteration split prevents the same stale external belief from being accumulated 15 times. Dynamics factors now converge before IR factors inject influence.
